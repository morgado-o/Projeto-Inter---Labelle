Projeto Integrador Interáreas - LaBelle

1- Descrição do Projeto
O projeto foi desenvolvido com o objetivo de promover a integração entre os alunos da Faculdade de Tecnologia de São José do Rio Preto (Fatec Rio Preto) e a comunidade externa. Por meio deste software, buscamos atender de forma eficiente às necessidades do setor, oferecendo uma solução tecnológica prática e acessível que contribui para a melhoria da gestão em clínicas e centros estéticos.

2- Funcionalidades do sistema
Nosso programa é um sistema de agendamento desenvolvido para atender clínicas e centros estéticos. Ele oferece uma solução prática e eficiente para organizar e gerenciar horários, otimizando o atendimento e facilitando a administração. O sistema foi projetado para atender às necessidades específicas do setor, proporcionando maior controle, economia de tempo e uma experiência mais satisfatória para os clientes e profissionais.

O sistema possui funcionalidades de: 
Agendamento;
Criação e edição de usuário do sistema;
Cadastro de procedimentos e clientes;
Agendamento de procedimentos por paciente;
Calendário com os agendamentos do cliente.


3- Tecnologia utilizadas

As linguagens de programação que utilizamos para a construção do projeto, foram:
Back-end: Java e SpringBoot
Front-end: JavaScript, HTML5 e CSS3
Banco de dados: SQL Server
